
# Expense Tracker

A full-stack Expense Tracker built using the PERN Stack with Prisma ORM and TypeScript.

## Live Demo

Frontend: https://expense-tracker-rho-green-43.vercel.app/

Backend: https://expense-tracker-11bw.onrender.com/

Repository: https://github.com/dexansh69/Expense-Tracker

---

## Features

- JWT Authentication
- Add Expenses
- Edit Expenses
- Delete Expenses
- Budget Management
- Dashboard Analytics
- Category-wise Pie Chart
- Search & Filter Expenses
- Responsive Design
- REST API
- PostgreSQL Database with Prisma ORM

## Installation

```bash
git clone https://github.com/dexansh69/Expense-Tracker.git

cd Expense-Tracker

npm install
```

Run frontend

```bash
npm run dev
```

Run backend

```bash
npm run dev
```

---

## Environment Variables

```env
DATABASE_URL=your_database_url

JWT_SECRET=your_secret
```

## API Endpoints

### Authentication

- POST /signup
- POST /signin

### Expenses

- GET /expense
- POST /expense
- PUT /expense/:id
- DELETE /expense/:id

### Budget

- POST /budget
- GET /stats

### Analytics

- GET /analytics
